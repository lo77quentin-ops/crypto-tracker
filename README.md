# Crypto Tracker Pro

Projet personnel développé avec Flask, Python, HTML/CSS et Bootstrap.

## Fonctionnalités

* Affichage des principales cryptomonnaies
* Graphique Bitcoin sur 7 jours
* Météo en temps réel :

  * Paris
  * Marseille
  * Lyon
  * Nantes
* Interface responsive
* Déploiement sur Render

## Technologies utilisées

* Python
* Flask
* HTML
* CSS
* Bootstrap
* Chart.js
* Open-Meteo API
